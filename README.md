# 音域確認表

React.js + Redux Toolkit + TypeScript + ESLint + Prettier の練習で作った音域確認表です。
DTM でオーケストラ風の曲を作りたい時に各楽器の音域がさっと確認できたら便利だなと思って作りました。

- 左側でチェックのついている楽器の音域が表示されます。
- 鍵盤をクリックやドラッグで選択するとその音を出せる楽器が表示されます。
